# Contributing to Image-to-PDF

First off, thank you for considering contributing to Image-to-PDF! It's people like you that make open source such a great community.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](../../issues) to see if someone else has already created a ticket. If not, go ahead and make one!

## Fork & create a branch

If this is something you think you can fix, then fork Image-to-PDF and create a branch with a descriptive name.

## Implementing your fix or feature

At this point, you're ready to make your changes! Feel free to ask for help if you need it.
Please ensure you stick to the vanilla JS conventions we've established.

## Make a Pull Request

At this point, you should switch back to your master branch and make sure it's up to date with Image-to-PDF's master branch.
Then create a pull request with a clear list of what you've done. When you submit a PR, we will review it and merge it!
